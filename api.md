# API utilizzate

Il progetto utilizza le API di Open-Meteo Forecast API per ottenere dati meteo da mostrare nella pagina.

-  https://api.open-meteo.com/v1/forecast

## Dettagli API

**Meteo**

- Meteo attuale: fornisce le condizioni meteorologiche in tempo reale per una posizione geografica specificata tramite coordinate.


**Previsioni**

- Previsioni orarie: tutti i parametri disponibili con unità di misura
- Previsioni giornaliere: riepilogo giornaliero fino a 7 giorni

## Note

- Le API di Open-Meteo sono gratuite per uso non commerciale
- I dati si aggiornano ogni ora circa
- Per dati storici è disponibile l'endpoint separato: https://archive-api.open-meteo.com/v1/archive
